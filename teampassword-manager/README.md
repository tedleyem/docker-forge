### Execute the Team Password Manager installer
To conclude the installation of Team Password Manager you should execute the Team Password Manager installer. To do this:

• Open your browser and go to [http://localhost/index.php/install](http://localhost/index.php/install). You can replace "localhost" with the IP address of your server or the domain/subdomain that points to it, if you have it configured. If you changed the ports in the docker-compose.yml file you should use this url: [http://localhost:8080/index.php/install](http://localhost:8080/index.php/install). Note also that we've used "http" and not "https" in this case. We explain how to configure https in this document: "How to install a https commercial certificate in a Docker container".
