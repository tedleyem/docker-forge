# SNIPE-IT 
Open source inventory management app. Showcase a use case for using SnipeIT and methods to enhance usage.

* [Installation docs](https://snipe-it.readme.io/docs/docker)

##### NOTES about .env file 
The .env file in this repo is exposed and should be used for testing purposes only. You can find an example [here](https://snipe-it.readme.io/docs/configuration) but should change it in your environment and hide it from source control. 

**The Challenge**
* Build out Snipe IT
* Create upgrade script to update versions of snipeIT 
* Create mysql backup script 
* Write a script to update inventory and assets through automation
